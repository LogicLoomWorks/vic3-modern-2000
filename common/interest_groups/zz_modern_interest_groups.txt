# Modern Interest Groups — vic3-modern-2000
# All 8 definitions use VANILLA IG key names to override vanilla IGs
# rather than adding new ones (adding new IGs triggers a hardcoded
# ExpectedNumItems assertion crash in CInterestGroupTypeDatabase).
#
# Mapping from design concept to vanilla key:
#   ig_tech_industry        → ig_industrialists
#   ig_military_establishment → ig_armed_forces
#   ig_religious_institutions → ig_devout
#   ig_organized_labor      → ig_trade_unions
#   ig_financial_sector     → ig_petty_bourgeoisie
#   ig_green_movement       → ig_landowners
#   ig_rural_populists      → ig_rural_folk
#   ig_intelligentsia       → ig_intelligentsia  (same key)
#   ig_security_services    → merged into ig_armed_forces (dropped as separate IG)
